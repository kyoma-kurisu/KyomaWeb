## saucenao

SauceNAO 是一款反向图片搜索引擎。其名称 “SauceNAO” 源于网络俚语 “Need to know the source of this Now!”（“立刻知道这张图的出处”）的缩写变形，这一说法在图片讨论版及同类平台中十分常见。
该网站由 Xamayon 设计并开发，核心搜索功能于 2008 年 10 月首次公开。最初的索引库仅包含约 2 万张杂志页面，使用的是基于 piespy 为 IQDB 项目开发的引擎修改版。随着访问量的持续增长和图片库规模的不断扩大，我们着手开发了更具可扩展性的自研引擎，并于 2012 年正式替换掉 IQDB 后端。自此，SauceNAO 进入高速发展阶段，目前已从各类网络平台、动画、漫画、电影、电视剧及其他多种来源中，索引了超过数十亿张图片。
SauceNAO 始终处于持续完善的状态。网站功能在不断迭代，内容覆盖范围在持续拓展，各类新型搜索技术也在不断试验中。我们觉得，这些尚未打磨完美的小瑕疵，也是它的独特魅力所在～ 

```
https://saucenao.com/
```

## Dialogue

Dialogue.moe 是一个开源项目，旨在帮助用户检索动画中的台词内容。项目中的台词均来自从互联网上爬取的大量中文字幕文件。这些台词通过 Elasticsearch 进行存储与分析，以支持全文检索功能。截至目前，数据库中已收录 7700 余部带字幕的动画，以及 650 万余条台词。由于服务器成本有限，当前仅配置了 2 核 vCPU 与 4GB 内存。请勿对本网站进行高并发爬取操作。

```
https://dialogue.moe/about
```

## trace

trace.moe 是一款动漫场景搜索引擎，用户只需上传截图，即可追溯其出处。 该引擎可在约 3 万小时 的动漫内容中进行检索，精准定位匹配度最高的场景，同时告知用户对应的动漫名称、集数以及该场景出现的精确时间。考虑到搜索结果可能存在误差，平台还提供了数秒的片段预览功能，方便用户验证结果。如今网络上流传着大量未标注出处的动漫截图与动图，而 trace.moe 的开发初衷正是为了改变这一现状 —— 帮助人们了解作品的完整出处，而非仅仅将其视为内容农场中不知来源的片段。

```
https://trace.moe/
```
