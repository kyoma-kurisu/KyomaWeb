## 直接插入图片
```
![显示的链接名字](XXX.com)
```

## 直接插入链接
```
[显示的链接名字](XXX.com)
```

## 实现折叠和展开功能
```
<details>
  <summary>点击展开/折叠</summary>

XXX

</details>
```

## 引用语法
```
>
>
>
```

## 标题
```
# 一级标题
## 二级标题
### 三级标题
```

## 正文
```
<div style="font-size: 18px;">

xxx

</div>
```

## 删除线
```
~~XX~~
```
## 表格语法
```
| 表头 1 | 表头 2 | 表头 3 |
| :--- | :----: | ---: |
| 左对齐内容 | 居中对齐内容 | 右对齐内容 |
| 内容 A | 内容 B | 内容 C |
```
