## 安装 Node.js 和 pnpm

访问 [Node.js](https://nodejs.org/en/download) 官方选择平台进行安装。

## 创建项目
```
pnpm create fuwari@latest
```

## 配置

编辑 src/config.ts 文件进行博客配置。

## 启动本地开发服务器

```
pnpm dev
```
## 创建新文章

使用命令
```
pnpm new-post <filename> 
```
或者直接在 src/content/posts/ 目录中创建文件。
## 部署
编译站点
```
pnpm build
```
