# steam批量挂刀教程
```
https://steamcommunity.com/market/multisell?appid=730&contextid=2&items[]=
```
## 在=号后面添加物品链接

![网页显示](steam挂刀教程1.jpg )

## 比如梦魇武器箱在社区市场的链接为：
```
https://steamcommunity.com/market/listings/730/Dreams%20%26%20Nightmares%20Case
```
### 其中
```
Dreams%20%26%20Nightmares%20Case
```
### 为梦魇武器箱的专属名

### 于是
### 在=号后添加**Dreams%20%26%20Nightmares%20Case**
```
https://steamcommunity.com/market/multisell?appid=730&contextid=2&items[]=Dreams%20%26%20Nightmares%20Case
```

### 即可实现批量售卖梦魇武器箱
